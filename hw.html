1. If we put display:inline property for div tag then will it become a span tag and vice versa for span tag? No, applying display: inline to a div tag does not turn it into a span tag, nor does applying display: block to a span tag turn it into a div. Here's why: 1. div and span Differences div: A block-level element by default, used for grouping content and structuring a webpage. span: An inline-level element by default, used to style or manipulate small chunks of text or inline content. 2. The Role of the display Property The display property only affects how an element is rendered in the layout (inline or block) but does not change the element's type or its semantic meaning. For example: A div with display: inline behaves like an inline element (it does not start on a new line and takes up as much width as necessary), but it is still a div in terms of HTML structure. A span with display: block behaves like a block element (it starts on a new line and takes up the full width available), but it is still a span. 3. Example: html Copy code
